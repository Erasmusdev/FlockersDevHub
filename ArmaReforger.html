Install & Host Your Own Server in Minutes

This script automates the setup of your Arma Reforger dedicated server — from downloading the files to configuring the basics.

Join Discord for Support

Known Bug('s)

  • Mods are not fully downloading (fix is comming)

Features

  • Downloads and installs SteamCMD automatically
  • Fetches and updates Arma Reforger server files
  • Generates configs/config.json with default settings
  • Creates a start_server.bat launcher
  • Adds Windows Firewall rules (UDP/TCP)

NEW: Local Control panel to control your server even easier

Requirements

  • Windows 10 or newer
  • Internet access (for SteamCMD and updates)
  • Download Visual C++ Redistributable (x64): https://aka.ms/vs/17/release/vc_redist.x64.exe
  • Router access is required (for public server hosting)
  • View the official server config documentation: https://community.bistudio.com/wiki/Arma_Reforger:Server_Config#top-page

Installation Steps

  1. Download and extract the installer package
  2. Install the VC_redist.x64 if not already installed
  3. Right-click install_server.bat → Run as Administrator
  4. The script will:
    • Download and extract SteamCMD
    • Install or update the server files
    • Create configs/config.json (if missing)
    • Generate start_server.bat
  5. Edit configs/config.json to match your preferences
  6. Launch the server using start_server.bat

Start_Server.Bat

This is the requred server command line: 

@echo off
cd /d "%~dp0"
"YOUR-FOLDER-LOCATION\arma_reforger_server\ArmaReforgerServer.exe" -config ".\configs\config.json"
exit /b

Port Forwarding

To host your server publicly, open these ports on your router and forward them to your server’s local IP address:

  • Game: UDP 2001
  • A2S Query: UDP 17777
  • RCON: TCP & UDP 19999
