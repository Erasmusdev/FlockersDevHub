Fun Chaos Effects for Your Stream

[FD] StreamMayham is a toolkit that lets your Twitch viewers trigger wild and chaotic effects on your PC through Streamer.bot or browser-based triggers.

With effects like mouse spins, fake crashes, input blocking, and more, StreamMayham brings interactive entertainment and viewer engagement — safely and externally without interfering with games or anti-cheat systems.

Key Features

  • Various chaos effects including mouse spins, fake crashes, input blocking
  • Triggered by Streamer.bot actions or web browser
  • Safe for use with popular anti-cheat systems
  • External applications, no game memory or code injection
  • Easy to configure and extend

How It Works

This toolkit uses separate C# applications that:

  • Simulate input (mouse movements, clicks)
  • Modify cursor behavior for effects
  • Trigger visual and audio chaos effects like fake crashes and jumpscares
  • Run externally, never modifying game processes or memory

Safe Use Guide

Designed with streamer safety in mind to avoid anti-cheat flags.

Safe Practices ✅ Avoid ❌
Run effects as separate .exe files Injecting code into game processes
Use Streamer.bot or web triggers to activate Writing memory of game process
Keep durations short and clearly triggered Emulating precise aim or timing macros
Visible/audible effects to show intent Masking actions to look like cheating

Compatible with popular anti-cheats including: EasyAntiCheat (Rust, Apex Legends), BattlEye (PUBG), and safe with games without anti-cheat like Paranormal Cleanup.

Setup Overview

  1. Set your effects `.exe` files to run as administrator.
  2. Create new Streamer.bot actions to trigger the desired effects.
  3. Use channel points, bits, or commands as triggers.
  4. In subactions, add Core > System > Run program and select the effect executable.

FAQ

Is this safe to use with anti-cheat systems?

Yes! StreamMayham effects run externally and do not modify game memory or inject code, making them safe with popular anti-cheats like EasyAntiCheat and BattlEye.

Can viewers trigger effects via browser?

Yes, you can trigger effects using a web browser-based trigger system alongside Streamer.bot.

Do effects require admin rights?

Some effects like input blocking or audio mute may require you to run executables as administrator.
