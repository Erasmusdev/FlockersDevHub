SERVER RULES

CODE OF CONDUCT

RULES

1. No Cheating/Hacking: Zero tolerance policy. Permanent ban on first offense.

2. No Racism/Harassment: Treat all players with respect. Toxic behavior will result in immediate action.

3. No Blocking/Griefing: Don't block monuments, resources, or spawn points. PVP is encouraged, griefing is not.

4. No Exploiting: Report bugs, don't abuse them. This includes roof/foundation camping exploits.

5. Team Limit: Maximum of 4 players per team. No alliances or teaming beyond limit.

6. Raiding Rules: Raid anyone, anytime. Counter-raiding is allowed. No raid rules except no exploiting.

7. KOS Allowed: Kill on sight is permitted. This is a PVP server - expect combat anywhere.

8. English Only: Primary language in global chat is English.

9. Admin Respect: Listen to admin instructions. Admins do not participate in PVP.
