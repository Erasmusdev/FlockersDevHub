🎯 Simply Ko-fi Goal Widget

Perfect for spooky streams, charity events, and fun audience engagement.

🎁 Join Discord

Easy Goal For Ko-fi V1 is a customizable and NON animated donation tracker build for creators using Ko-fi. Integrates with Streamer.bot or browser triggers for rich interactions.

✨ Key Features

  • 🎨 Fully customizable color palette
  • ⚙️ Easy to configure with Streamer.bot
  • 📊 Live progress Update
  • 🧰 Lightweight and simple visual style
