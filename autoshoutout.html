What is AutoShoutout?

AutoShoutout is a Streamer.bot automation that detects when a Twitch streamer enters your chat and automatically plays a customizable overlay, audio cue, and typed message ‚Äî all hands-free.

Installation Overview

  1. Download the AutoShoutout .zip package
  2. Import the `.sba` file into Streamer.bot
  3. Connect your Twitch account in Streamer.bot
  4. Customize overlay text and audio
  5. Enable the action trigger: On Twitch Chat Message

Required Software

  • Streamer.bot (latest version)
  • OBS Studio (with WebSocket plugin)
  • Twitch account connected to Streamer.bot

Setting Up the Action

  1. Open Streamer.bot
  2. Go to Actions > Import > Select the provided `.sba` file
  3. Edit the action to personalize overlays or add audio
  4. Ensure OBS WebSocket is enabled and authenticated
  5. Test with a friend or an alt Twitch account

Customization Tips

  • Add sound effects using OBS Source Toggle or Play Sound
  • Use the {user} tag for dynamic usernames
  • Combine with OBS filters for animations
  • Create a whitelist of VIPs/Streamers if needed

FAQ

Does this require OBS?

Yes, OBS is required if you're using overlays. Audio-only shoutouts can work without it.

Can I control who gets a shoutout?

Yes, you can edit the C# subaction to check against a list of usernames, roles, or categories.

Will it repeat if someone chats multiple times?

No. It only triggers once per user per stream session unless you reset the tracking variable manually.
